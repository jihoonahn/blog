# Swift로 시작하는 웹 개발

Swift는 iOS 개발뿐만 아니라 서버 사이드 개발에서도 강력한 도구입니다.

## 시작하기

먼저 프로젝트를 생성합니다:

```bash
swift package init --type executable
```

이제 여러분의 첫 Swift 웹 애플리케이션을 만들 준비가 되었습니다!
