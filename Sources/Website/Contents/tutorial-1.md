# Swift 기초 튜토리얼

Swift 프로그래밍 언어의 기초를 배워봅시다.

## 변수와 상수

Swift에서는 `var`로 변수를, `let`으로 상수를 선언합니다.

```swift
var greeting = "Hello"
let name = "Swift"
```

## 함수

함수는 `func` 키워드로 선언합니다.

```swift
func greet(person: String) -> String {
    return "Hello, \(person)!"
}
```

이것이 Swift의 기본입니다!
